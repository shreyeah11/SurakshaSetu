# 🛡️ SurakshaSetu — A Bridge to Digital Safety

SurakshaSetu is a **cybersecurity training and protection platform** designed to help users identify, understand, and avoid digital scams through **interactive learning, AI-powered simulations, and real-time analysis tools**.

---

## 🚀 Problem Statement

As the world rapidly transitions to **digital-first services** (banking, healthcare, government platforms), a large segment of users—especially beginners and elderly individuals—struggle to distinguish between **legitimate services and sophisticated scams** such as phishing, OTP fraud, and deepfakes.

Existing solutions focus mainly on detection, but lack **practical training and real-world simulation**, leading to continued vulnerability.

---

## 💡 Our Solution

SurakshaSetu bridges this gap by combining:

* 🎮 **Interactive Learning**
* 🤖 **AI-Powered Scam Simulation**
* 🛠 **Real-Time Detection Tools**
* 🚨 **Community-Driven Alerts**

It provides a **sandbox-style environment** where users can safely learn and practice identifying cyber threats.

---

## ✨ Key Features

### 🎮 1. Interactive Learning Modules

* **Red Flag Hunter** — Identify suspicious elements in phishing messages
* **Security Quiz** — Test cybersecurity awareness
* **Spot the Difference** — Compare real vs scam messages
* **Deepfake Recognition Lab** — Learn to detect AI-generated media

---

### 🤖 2. AI Scam Simulator *(Core Innovation)*

* LLM-powered chatbot that **roleplays as a scammer**
* Users interact via guided choices (safe environment)
* Provides **real-time feedback and risk scoring**
* Simulates tactics like urgency, fear, and manipulation

---

### 🛠 3. Cyber Safety Toolkit

* **Spam Detector** — Analyze suspicious text or files
* **Deepfake Detector** — Identify AI-generated images/audio
* **Message Analyzer** — Highlight phishing patterns and risks

---

### 🚨 4. Community Siren (Crowdsourced Alerts)

* Users can **report real-world scams**
* Generates a live **community feed**
* Highlights **“Warning of the Day”** based on reports
* Encourages collaborative digital safety

## 🏗️ System Architecture

SurakshaSetu follows a **modular and scalable architecture**:

### 🔹 Frontend Layer

* Built with **React**
* Handles UI, simulations, and user interactions

### 🔹 Processing & Logic Layer

* Handles user inputs and decision-making logic
* Includes rule-based validation and scoring systems

### 🔹 Intelligent Detection Layer

* Integrates **AI/LLM models** for scam simulation and analysis
* Performs pattern recognition and risk evaluation

### 🔹 Data & Support Layer

* Stores user data, scores, and community reports
* Supports real-time updates and alerts
